# Bash Tips

## Raccourci clavier

## Recherche 
 - `grep -rn "ref" . `:
	permet de trouver toutes les references de "ref" dans le dossier "."
	très pratique pour cherche la definitions d'une référence.

## Miscealinous
 
 - `sync`:
	synchronise la buffer disque avec la memoire.
	pratique si on fait xxd, (et autres dump d'un disque physique ou virtuel)
	ps: c'etait utile pour PNL. 
	
 - `cd -`:
 	permet de retrouner dans le dernier dossier avant le denier cd
	pratique pour voyager entre deux dossier, par exemple le dossier de compilation et 
	le dossier avec le fichier à executer.
